About Game

This game has a name from no moon day which is called Amavasya in India. People believe that in Amavasya supernatural things occurs which is related to paranormal activity.

What Makes It More Interesting

The most interesting thing about this game is that it is in Hindi, people love when they hear dialogues in their favorite language.
The game has doors that are locked and a player needs to find keys to open the doors and there is a ghost that protects the main door's keys. If ghosts touch you then the game will restart.The most dangerous things is ghost can even touch you near windows sometimes near walls because they are ghost and game have some glitches.
If you are a player who never gives up then you will like it very much this is a little bit hard if you are trying first time when you start finding keys then it will become easier for you to play.
There is a graveyard in this game in which if you stand on graves they will tell you their death story which make it more creapy.
